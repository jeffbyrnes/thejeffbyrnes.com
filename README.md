# Website of Jeff Byrnes

A simple, hand-written website. No build step: the contents of [`docs/`](docs/)
are served as-is (via GitHub Pages).

Design by [Dan Perrera & Amy Parker](https://perrera.com).

## Working on it

The CSS in [`docs/css/`](docs/css/) is plain, hand-authored CSS — edit it
directly. Linting is optional and only needs Node:

```sh
npm install
npm run lint
```
